# Nostra – Fashion E-Commerce Website

A fully responsive e-commerce website built using **HTML**, **CSS**, and **JavaScript**.

---

## 🌐 Live Demo

https://nagendrana.github.io/Nostra-Ecommerce/

---

## 📁 Project Structure

```
Nostra/
├── index.html        → Home Page
├── collections.html  → Collections Page (Search & Filter)
├── contact.html      → Contact Us Page
├── style.css         → Stylesheet for all pages
└── script.js         → JavaScript functionality
```

---

## 📄 Pages

### 1. Home Page (`index.html`)
- Offer bar with close button
- Sticky navbar with logo and navigation links
- Hero image slider with auto-play (3 slides)
- New Arrivals section (4 products)
- Most Wanted section (4 products)
- Collections section — Men, Women, Accessories
- Footer with quick links, categories, and newsletter

### 2. Collections Page (`collections.html`)
- Hero banner with page title
- **Live search** — filter products by name as you type
- **Filter buttons** — All, Men, Women, Accessories, Streetwear, Premium
- **Sort** — Default, Price Low to High, Price High to Low, Name A–Z
- 12 product cards with category tags and prices
- No results message with reset option
- URL parameter support (e.g. `?filter=men` from home page)

### 3. Contact Page (`contact.html`)
- Hero section with background image
- Info cards — Visit Us, Call Us, Email Us
- Two-column layout — Contact form (left) + Fashion image with quote (right)
- Form fields — First Name, Last Name, Email, Subject dropdown, Message
- FAQ accordion section — 4 questions with open/close animation
- Full footer with newsletter signup

## 🚀 How to Run

1. Clone or download the repository
2. Open `index.html` in any browser
3. No installations or dependencies needed

```bash
git clone https://github.com/your-username/nostra-ecommerce.git
cd nostra-ecommerce
# Open index.html in browser
```

## 📱 Responsive Breakpoints
